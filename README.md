# Smart NGO Resource Allocation System

## 🚀 Overview

An AI-powered platform that connects community problems with the right volunteers in real time using location, skills, and smart matching.

## 🎯 Problem

NGOs face inefficient resource allocation, lack of real-time data, and delayed response to community issues.

## 💡 Solution

* Collects community issues
* Displays them on a live map (hotspots)
* Matches volunteers based on location & skills
* Sends instant notifications

## 🧠 Key Features

* Problem Reporting (with geo-tagging)
* Live Map Dashboard
* Smart Volunteer Matching
* Real-Time Notifications
* Task Tracking

## 🏗️ Tech Stack

* Frontend: React.js
* Backend: Node.js / Express
* Database: MongoDB
* APIs: Maps & Notifications

## 🌍 Impact

Faster response, better coordination, and improved NGO efficiency.
